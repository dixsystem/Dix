FAQ

Before you ask.

The answers to everything.

Is it safe? Will it break my system?
Yes, it's safe. Completely. DIX only modifies sysctl parameters and CPU governors — all within the boundaries defined by the Linux kernel itself. We never touch GPU settings, never modify NUMA topology, and never apply changes that can't be reverted. Every parameter change is validated before being applied.
What Linux distributions are supported?
DIX works on any Linux distribution with kernel 5.4+. This includes Ubuntu, Debian, Arch Linux, Manjaro, Fedora, openSUSE, Pop!_OS, Linux Mint and more. If your distro runs systemd and has a modern kernel, it works.
Do I need root / sudo access?
Some kernel parameters require elevated permissions to apply. DIX will prompt for authentication via pkexec (PolicyKit) when needed — the standard secure way Linux applications handle privileged operations. You remain in full control.
What happens after I buy? Is it really one-time?
Yes. One payment. Forever. After purchase via Lemon Squeezy you receive a license key and download link by email. The key activates the application. You get all v1.x updates for free. No subscription, no renewal, no expiry. Windows and macOS versions will be separate purchases (or a bundle deal — TBA).
Does it send my data anywhere?
The AI analysis is performed via a call to Claude API using the system data collected locally. No personal data is transmitted. No usage telemetry. No analytics. The only network call is the optimization analysis — and you see exactly what's being analyzed before it happens.
